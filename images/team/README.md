# Adding Team Member Photos

## 📸 Photo Specifications

- **Size**: 300x300 pixels (square) for optimal display
- **Format**: JPG or PNG
- **File size**: Under 500KB each
- **Quality**: Professional headshots work best

## 📁 File Naming Convention

Save photos with lowercase, hyphenated names matching the HTML:

### Staff:
- `tommy-ryan-keogh.jpg`
- `flavien-petit.jpg`
- `elise-droste.jpg`
- `orlando-prugel-bennett.jpg`

### Alumni:

## 📤 How to Upload

### Method 1: GitHub Web Interface

1. Go to your repository on GitHub
2. Navigate to `images/team/` folder
3. Click "Add file" → "Upload files"
4. Drag your photos here
5. Click "Commit changes"

### Method 2: Replace Locally

1. Add photos to this folder on your computer
2. Upload the entire `images/team/` folder to GitHub

## 🎨 Photo Preparation Tips

### Option 1: Online Tool (Easiest)
1. Go to https://www.iloveimg.com/crop-image
2. Upload photo
3. Select "Square" crop
4. Resize to 300x300
5. Download

### Option 2: Using Your Phone
- iPhone: Use built-in Photos app → Edit → Crop → Square
- Android: Use Google Photos → Edit → Crop → Square

### Option 3: Professional Tools
- Photoshop: Image → Image Size → 300x300px
- GIMP: Image → Scale Image → 300x300px

## ⚠️ What Happens Without Photos?

The placeholder image will show automatically if a photo is missing. The website won't break!

## 🔄 Updating Photos Later

Just upload a new file with the same name - it will automatically replace the old one.

## 💡 Pro Tips

- Use consistent backgrounds (solid color or blurred)
- Ensure good lighting
- Crop to show head and shoulders
- Save as JPG for smaller file size
- Compress images before uploading

## 🆘 Troubleshooting

**Photo not showing?**
- Check filename matches exactly (case-sensitive)
- Ensure it's in `images/team/` folder
- Clear browser cache (Ctrl+F5)
- Wait 1-2 minutes for GitHub Pages to update
